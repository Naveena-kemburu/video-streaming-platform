# Demo Video Script (2-4 minutes)

## Introduction (15 seconds)
- Show the landing page with video library
- Highlight the responsive design and modern UI

## Core Features (60 seconds)

### Video Library
- Demonstrate search functionality (search for "Buck")
- Show category filtering (click Animation, Sci-Fi)
- Point out lazy-loaded thumbnails

### Video Playback
- Click on a video to navigate to detail page
- Show the video player loading and starting playback
- Demonstrate adaptive bitrate streaming indicator

## Player Controls (45 seconds)

### Basic Controls
- Play/Pause button
- Volume control and mute
- Seek bar (drag to different positions)
- Fullscreen toggle

### Advanced Features
- Playback speed control (0.5x to 2x)
- Quality selector (auto, 1080p, 720p, 480p)
- Subtitles/captions toggle
- Picture-in-Picture mode

## Keyboard Navigation (30 seconds)
- Press Space to play/pause
- Press M to mute
- Press F for fullscreen
- Arrow keys for seeking and volume
- Show focus indicators

## User Features (30 seconds)
- Add video to favorites (heart icon)
- Navigate back to library
- Show "Continue Watching" section with progress bar
- Click another video to show progress persistence

## Performance Demo (30 seconds)
- Open Chrome DevTools
- Navigate to Network tab
- Throttle to "Fast 3G"
- Show adaptive bitrate switching in action
- Video continues playing smoothly

## Accessibility (15 seconds)
- Tab through all controls
- Show ARIA labels in DevTools
- Demonstrate screen reader compatibility (optional)

## Conclusion (15 seconds)
- Show Lighthouse scores (Performance, Accessibility, Best Practices > 90)
- Thank you message
